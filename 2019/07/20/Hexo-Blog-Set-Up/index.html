Hexo Blog Set Up

Posted on 2019-07-20 Comments:

Set up blog with Hexo, Next theme, github.

Blog set up

Install Hexo

Create blog dir

After installation, _config.yml is important under dir. Basic setting of website, e.g., title, description, keywords, etc.
Finish config change, in blog dir, exec

local mode has been started. Can be accessed through http://localhost:4000

Integete with github

github provided github.io to enable website build. Create a public repo with your ${github_account}.github.io.

addon for hexo

Install hexo-deployer-git.

Configure your github repo in _config.yml

deploy blog to github repo

exec

Access your new website

Be exited to see website through https://${github_account}.github.io
